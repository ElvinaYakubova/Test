Таблица article – статьи, содежит уникальный id статьи, название, id автора, дату создания, дату последнего обновления, id категории, превью и сам текст статьи.
Таблица category – категории статей,  связана с таблицей article полем id категории.
Taблица tag содержит названия тэгов для статей, связана с таблицей article через промежуточную таблицу art_tag.
Таблица users содержит информацию о пользователях, поле role хранит информацию о правах доступа (2 – админ,  1 – пользователь с правом написания статей, 0 – гость без права написания статьи) 
Таблица comment – комментарии к статьям, содержит id комментария, id пользователя, оставившего комментарий, id статьи, к которой относится коммент, сам текст , дату и время отправки, id родительского коммента и т.п
В таблицу  fav article заносятся только статьи, добавленные в избранное конкретным пользователем. 
В таблицу like_dislike заносятся статьи или комментарии, которые лайкнул определенный пользователь (1-like, 0-dislike). Поле object_id соответствует id коммента или статьи, а поле type определяет тип (т.е был ли это id комментария или id статьи) .
